# redgoose-v4

redgoose.me ver.4

## 설치하기

저장소를 클론 받으면 다음 명령어로 실행해줘야함.

```bash
composer install
yarn install
yarn run setup
```

## 개발하기

```bash
yarn run start # 개발서버 시작 (php)
yarn run watch # css, js watch
```

## 배포하기

```bash
yarn run build
```
